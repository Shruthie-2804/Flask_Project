# Flask API Project

This project is a simple Flask-based REST API.

## Features
- Health check API
- Echo API using POST request
- Add API using query parameters

## Requirements
- Python 3.x
- Flask

## How to Run
1. Download the repository as ZIP
2. Extract the files
3. Open command prompt in the project folder
4. Install dependencies:
   pip install flask
5. Run the application:
   python app.py

## API Endpoints
- GET /health
- POST /api/v1/echo
- GET /api/v1/add?a=10&b=20
